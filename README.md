
# 📊 Sales Performance Dashboard | Power BI | Excel

## 🔍 Overview
This project focuses on analyzing sales data to uncover meaningful business insights using Power BI.  
The dashboard provides a clear view of sales performance, customer behavior, and product trends to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing products
- Track monthly and yearly sales trends
- Understand customer purchasing behavior
- Provide actionable insights for business growth

---

## 📁 Dataset Details
The dataset includes the following key fields:
- Order Number
- Order Date (Year, Month, Quarter)
- Product Line & Product Code
- Customer Name
- Quantity Ordered
- Price Each
- Sales Amount
- Deal Size / Status

---

## 📊 Key Insights
✔ Identified top revenue-generating products  
✔ Analyzed monthly sales trends to detect peak seasons  
✔ Highlighted high-value customers contributing to revenue  
✔ Compared sales performance across product categories  
✔ Tracked order status and deal distribution
